========================================================================
                      Bwenke Educational Website
========================================================================

A modern, professional, fully responsive multi-page educational website 
specifically developed using semantic HTML5, custom CSS3 properties, 
and vanilla ES6+ JavaScript.

------------------------------------------------------------------------
1. File Structure
------------------------------------------------------------------------
/ (Workspace Root)
├── index.html          - Home Page
├── courses.html        - Courses & Programs Catalog
├── chemistry.html      - Pure Science: Chemistry Syllabus, Notes, Lab
├── biology.html        - Life Science: Biology Syllabus, Diagrams, Tips
├── cbc.html            - CBC Curriculum Framework & Core Competencies
├── about.html          - Origin Story, Vision, Team, & Contact Form
├── css/
│   └── style.css       - Clean, shared stylesheet with custom CSS vars
├── js/
│   └── script.js       - Single interactivity & animation script
├── metadata.json       - Applet configuration
├── vite.config.ts      - Multi-Page Application bundle mapping
└── README.txt          - Project Documentation (This file)

------------------------------------------------------------------------
2. Core Technologies Used
------------------------------------------------------------------------
- HTML5: Semantic structural tags (<header>, <nav>, <section>, <footer>)
- CSS3: Custom properties, CSS Grid, Flexbox, media queries
- Vanilla JavaScript (ES6+): Single interactive engine
- Google Fonts: "Inter" for body copy, "Space Grotesk" for display text,
                and "JetBrains Mono" for statistical parameters

------------------------------------------------------------------------
3. Interactive Features (JavaScript Powered)
------------------------------------------------------------------------
1. Scroll Reveal Animations: Automatic sliding and fade-ins triggered 
   via IntersectionObserver as items slide into the active viewport.
2. Animated Counters: Statistics increment from 0 to target metrics 
   upon scroll visibility.
3. Typing Animation: Elegant dynamic keyword typing loops on the home page.
4. Mobile Navigation Toggle: Responsive hamburger menu with seamless drawer transitions.
5. Back to Top Button: Floating circular control fade-in triggered by scroll position, supporting smooth scrolls.
6. Accordions: Multi-panel notes and revision material collapsible boxes.
7. Active Link Highlighting: Detects file URLs to highlight active links.

------------------------------------------------------------------------
4. Styling & Customization Themes
------------------------------------------------------------------------
The website implements a distinctive, high-end "Sapphire Corporate Blue" 
theme with clean rounded cards, soft drop shadows, generous line heights, 
and spacious margins. All icons and vector graphics are rendered natively 
using scalable vector SVGs to maintain crisp clarity on all retina screens.

Thank you for choosing Bwenke. Empowering minds, one discovery at a time.
